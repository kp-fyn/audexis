## Edit a Single File

1. Click a file
2. Use the sidebar to edit fields
3. Click **Save** when ready

## Edit Multiple Files

1. Select files (⌘+Click for multiple, Shift+Click for range)
2. Use the sidebar to edit fields
3. Click **Save** to write changes

Your changes apply to **all selected files**
**Example**: Select 10 songs, edit the Album field once, and all 10 files get the same album name.

## Common Tags

- **Title** — Song name
- **Artist** — Performer
- **Album** — Album name
- **Album Artist** — For compilations
- **Year** — Release year
- **Track Number** — Position in album
- **Genre** — Music style
- **Composer** — Who wrote the song

## Album Artwork

1. Select files
2. Double-click the **Attached Picture** column
3. Click **Import Image**
4. Choose a PNG or JPEG file
5. Save changes

**Tip**: Keep images under 1 MB for best compatibility.

## Quick Tips

- **Navigate while editing**: Press Tab to move between fields
- **Find and Replace**: Use ⌘+Shift+F for bulk text changes
- **Rename files**: Select files, Right Click, click rename using pattern, use pattern like `{artist} - {title}.{ext}`
- **Copy/Paste**: ⌘+C and ⌘+V work between cells

## Save Your Changes

Audexis doesn't auto-save by default:

1. Make your edits
2. A **Save Bar** appears
3. Click **Save** or press **⌘+S**
4. Changes are written to your files

To discard changes, click **Discard** or press Esc.
