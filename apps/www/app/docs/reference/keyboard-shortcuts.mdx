Quick reference for built‑in shortcuts.

## Global

- Cmd+I — Import files
- Cmd+Shift+I — Import folder
- Cmd+R — Refresh
- Cmd+F — Open Find
- Cmd+Shift+F — Open Replace
- Cmd+G — Next match
- Cmd+Shift+G — Previous match
- Cmd+Enter — Replace one
- Cmd+Shift+Enter — Replace all

## Selection

- Cmd+A — Select all files
- Esc — Clear selection (nudges the Save bar if unsaved changes)
