# Appearance

Customize how Audexis looks.

## Theme

Choose **Light** or **Dark** mode.

Changes apply immediately. You can also toggle the theme from the navbar.

## Density

Control the spacing of UI elements:

- **Compact** — Tighter spacing, see more files
- **Default** — Balanced
- **Comfort** — More space, easier on the eyes

## Open Settings

Press **⌘+,** anytime to access settings.
