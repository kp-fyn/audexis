# Columns

Choose which metadata fields appear in the table.

## Show/Hide Columns

Toggle the checkbox next to each field to add or remove it from the table.

## Reorder

Drag the handle (â‰¡) to change the order. Changes save automatically.

You can also drag column headers in the main table.

## Search

Use the search box to quickly find specific fields.

## Common Fields

- Title, Artist, Album, Year
- Track Number, Genre
- Album Artist, Composer
- Attached Picture (album art)

## Resize Columns

In the main table, drag the edge between headers to adjust width.
