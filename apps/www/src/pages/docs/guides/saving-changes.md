# Saving Changes

Understand how Audexis handles save workflows.

## Save bar

When you have unsaved edits, a Save bar appears. Use it to commit or discard changes.

Press Esc to clear selection. If there are unsaved edits, the app nudges the Save bar instead so you don’t lose work.
